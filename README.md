# Arduino-Tetris
 Tetris game using components from popular arduino kits

# Components list
1) Led matrix 8x8 (one colored)
2) 8-bit shift register 74HC595
3) 8x 220hOm resistors
4) 1x 10k resistor 
5) Arduino Uno r3
6) Connectors
